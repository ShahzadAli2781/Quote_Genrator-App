# Random Quote Generator ✨

A premium and visually stunning Random Quote Generator Android application developed in Java using Android Studio as part of the CodeAlpha App Development Internship.This app combines luxury-inspired Glassmorphism UI, cinematic animations, motivational content, and modern Material Design principles to deliver a smooth and elegant user experience.The application was designed to feel like a real Play Store productivity and motivation app while keeping the internal logic beginner-friendly and easy to understand.

# 🌟 App Highlights

* Premium Glassmorphism UI
* Smooth Cinematic Animations
* Luxury Purple-Lavender Theme
* Beautiful Motivational Quote Experience
* Authentication System (Login & Signup)
* Settings & Profile Screens
* Share Quote Feature
* Modern Material Design Components
* Beginner-Friendly Java Logic
